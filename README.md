# AoC Solutions for 2025

How to use:

```
usage: aoc.py [-h] [-t] day

positional arguments:
  day         Number of day

options:
  -h, --help  show this help message and exit
  -t, --test  Run on test data
```

Example for running day 1 on test data

```
python aoc.py 1 -t
```

Example for running day 1 on final data

```
python aoc.py 1
```
